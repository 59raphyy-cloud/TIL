### 기초 수식 1
> 다음 재귀식들을 빅오 표기법 수준으로 풀이하시오.

문제 1: T(n) = T(n-1) + 1, T(0) = 1

```
T(n) = T(n-1) + 1
     = T(n-2) + 1 + 1
     = T(n-k) + k
     = T(0) + n, n == k
     = 1 + n

O(T(n)) = O(n)
```

### 기초 수식 2
> 다음 재귀식들을 빅오 표기법 수준으로 풀이하시오.

문제 1: T(n) = T(n-1) + n, T(0) = 1

```
T(n) = T(n-1) + n
     = T(n-2) + (n-1) + n
     = T(n-k) + (n-k+1) + ... + n
     = T(0) + 1 + ... + n, n == k
     = 1 + (1 + n)n / 2

O(T(n)) = O(n^2)
```
