## 비트 연산

- & : 둘 다 1이어야 1
- | : 하나만 1이어도 1
  
### XOR(Exclusive-OR) 연산자
- ^ : 둘 다 1이거나 0인 경우 0 (다른 경우만 1)
- 어떤 값이던 특정 수로 2회 XOR 하면 원래 수로 돌아옴

### 쉬프트 연산자
- Left Shift << : 특정 수 만큼 비트를 왼쪽으로 밀어냄
- Right Shift >> : 특정 수 만큼 비트를 오른쪽으로 밀어냄 (우측 비트 제거)

[응용]
```python
1 << n
```
- 2^n의 값을 가짐
- 특정 자리수의 값을 비교하고 싶을 때 & 연산자와 함께 활용
```python
i & (1 << n)
```
- i의 n번째 비트가 1인지 아닌지 확인
```python
arr = [7, 1, 3, 5]
1 << len(arr)
```
- 부분 집합의 수
---
[음수 표현 방법]
- 2의 보수: 각 비트를 뒤집고 +1 한다.
- 2의 보수를 한 번 더 취하면 원래 값으로 돌아온다.
---
### Bitwise NOT(complement) 연산자
- ~ : 모든 비트를 반전시킴
---

```python
t = 3.141592

```
